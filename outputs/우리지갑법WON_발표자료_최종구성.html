Problem Definition

청소년의 소비 경험은 많지만, 소비를 해석하는 피드백은 부족합니다

지출 인식 부족소액 반복 지출은 쉽게 누적되지만, 사용자는 자신의 소비 습관을 체감하기 어렵습니다.
목표 기반 절제 약함갖고 싶은 상품은 있지만, 소비 절제와 저축 목표가 서비스 안에서 연결되지 않습니다.
금융 교육의 낮은 몰입도일방향 금융 교육은 행동 변화로 이어지기 어렵고, 개인 소비 맥락과도 멀어집니다.
4/22
Service Overview

우리카드 앱을 슈퍼앱으로 확장하는 소비 코칭 마이크로서비스

Goal목표 상품 금액을 중심으로 사용자의 절제 동기를 만듭니다.
CourtMulti-Agent 재판 형식으로 지출을 분석하고 낭비 여부를 설명합니다.
RewardAI 콘텐츠와 티어 보상으로 금융 학습을 게임처럼 이어갑니다.

핵심은 기존 우리카드 앱 안에서 독립적으로 배포 가능한 우리지갑법WON 서비스를 붙여, 앱의 금융 경험을 교육·절제·보상으로 확장하는 것입니다.

MSA Separation

우리카드 앱과 우리지갑법WON은 API와 DB 책임을 분리했습니다

우리카드WON 앱 백엔드
  • 로그인, 사용자, 인증, MyData 동의 상태 관리
  • 우리카드 및 금융 상품 추천에 필요한 기준 데이터 제공
  • 인증/회원 중심 DB 책임
우리지갑법WON 서비스 백엔드
  • 목표 상품, 지출 분석, AI 재판, 판결, 포인트/티어 관리
  • AI 서비스와 mock 금융 데이터 연동
  • 소비/재판 중심 DB 책임

서비스 간 직접 DB 공유를 피하고, 사용자 식별자는 API 계약으로만 전달해 독립 배포와 장애 격리를 목표로 했습니다.

AWS EKS Infrastructure

Route53부터 EKS 파드까지 이어지는 실제 AWS 인프라

Route53dannis.cloud DNS
ACMHTTPS 인증서
API Gateway서비스별 public endpoint
VPC Linkprivate VPC 연결
Internal ALB공유 내부 라우팅
EKS Service서비스별 트래픽 수신
Podsfrontend, backend, AI, DB, monitoring
ECRGitHub Actions가 빌드한 Docker 이미지를 저장하고 EKS가 실행 시점에 pull합니다.
SSM + ESOAPI 키와 DB Secret은 이미지에 넣지 않고 런타임 Secret으로 주입합니다.
Cost-aware DB비용 절감을 위해 현재 DB는 EKS 내부 StatefulSet과 PVC 기반으로 운영합니다.

Git Flow와 Argo CD로 배포 기준을 Git에 고정했습니다

develop PR GitHub Actions가 테스트와 이미지 빌드 검증을 수행해 기능 통합 전 품질을 확인합니다.
main PR 이미지를 ECR에 push하고, infra repo의 image tag commit을 통해 Argo CD 배포를 트리거합니다.

15/22
Operation

서비스 운영은 AWS 레이어와 관리자 레이어로 나누어 관리합니다

Credential 운영SSM Parameter Store와 External Secrets Operator로 민감 값을 배포 시점이 아니라 실행 시점에 주입합니다.
DB 운영도메인별 DB를 분리하고, PVC 기반 상태 저장 리소스로 데이터 생명주기를 관리합니다.
서비스 관제Prometheus, Grafana, Alertmanager로 파드 상태, 재시작, 서비스 메트릭을 관찰합니다.
관리자 화면사용자, 콘텐츠, 티어, 데이터 품질을 운영자가 확인할 수 있는 레이어를 둡니다.
할루시네이션 관리생성형 콘텐츠는 추적과 검수 기준을 통해 잘못된 판결·교육 콘텐츠를 줄입니다.
접근 제어Grafana 등 운영 도구는 관리자 IP 기반으로 공개 범위를 제한합니다.

Problem Solving Report

소비 습관 변화는 두 가지 대표 사례로 설명할 수 있습니다

사례 1. 충동 소비형 반복되는 배달·카페·편의점 지출을 재판과 피드백으로 자각하고, 목표 상품을 위해 줄일 수 있는 지출을 발견합니다.
사례 2. 목표 미설정형 막연한 저축이 아니라 원하는 상품을 중심으로 계획을 세우고, 티어와 퀴즈 보상으로 행동을 유지합니다.

Business Value

기업 입장에서는 교육, 전환, 장기 고객 관계를 함께 얻습니다

잠재 고객 확보청소년기부터 우리카드 앱 안에서 금융 경험을 축적하게 합니다.
